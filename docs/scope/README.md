# EasyFoods — Complete Product Scope

This folder documents everything the EasyFoods platform promises to deliver.
It is organized by actor and module. Each file defines the full scope of that area.

## Actors

The platform serves three distinct user groups, each with their own interface and needs:

| Actor | Interface | Primary goal |
|-------|-----------|-------------|
| Restaurant | Backoffice panel + Kitchen panel | Manage operations, products, staff, orders |
| Customer | Storefront + Order tracking | Browse, order, customize, track |
| Delivery Driver | Driver panel | Receive, complete, and update deliveries |
